pacman 引起的错误：
	1.Failed to commit transaction (conflicting files)" error
	pacman has detected a file conflict, and by design, will not overwrite
    files for you. This is a design feature, not a flaw.
        The problem is usually trivial to solve. A safe way is to first check if anoter package owns the file
	( pacman -Qo/path/to/file ).
